    Class InferentialStats

    InferentialStats - Main SDK class for browser-based statistical analysis.

    Uses Pyodide (WebAssembly) running in a Web Worker to perform SPSS-level statistical computations entirely client-side.

    Data is transferred to the worker using ArrayBuffer (Transferable Objects) for maximum efficiency with large datasets.
